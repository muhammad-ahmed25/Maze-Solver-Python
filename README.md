# Maze-Solver-Python
PROJECT NAME
Maze Game / Maze Solver – Python

------------------------------------------------------------

INTRODUCTION

This project is a Python-based maze program that demonstrates:

• Maze creation  
• Player movement / path finding  
• Logic implementation using Python  
• Use of functions and control structures  

The program can be used as a simple game or algorithm demonstration.

------------------------------------------------------------

FEATURES

✔ Display maze structure  
✔ User movement inside maze  
✔ Wall and path detection  
✔ Start and end point logic  
✔ Keyboard based controls  

------------------------------------------------------------

TECHNOLOGIES USED

• Python 3  
• Console based interface  
• Logical programming  

------------------------------------------------------------

HOW TO RUN

1. Make sure Python is installed  
2. Open terminal in project folder  
3. Run:

   python maze.py

------------------------------------------------------------

HOW TO PLAY / USE

• Run the program  
• Follow on-screen instructions  
• Use keyboard input to move  
• Reach the goal to finish maze  

------------------------------------------------------------

CONCEPTS USED

• Lists and arrays  
• Loops and conditions  
• Functions  
• User input handling  

------------------------------------------------------------

POSSIBLE IMPROVEMENTS

• Add GUI version  
• Add auto solver algorithm (BFS/DFS)  
• Add multiple levels  
• Add timer and score
